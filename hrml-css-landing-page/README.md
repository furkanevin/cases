# Social Care Wales – Front-End Task

This repository contains a standalone front-end page developed as part of the Social Care Wales front-end developer task.

The goal of this task was to recreate the provided design as closely as possible while producing clean, semantic, and responsive HTML and CSS.

---

## 📱 Responsiveness

The layout is responsive and adapts to different screen sizes using:

- CSS Grid and Flexbox
- Responsive spacing and layout adjustments

---

## 🎨 Design & Typography

- The design was implemented based on the provided Figma file.
- The typeface used is **Gibson** where available, with **Arial** as a fallback, in line with the task instructions.
- Colours, spacing, and layout were matched as closely as possible to the original design.
